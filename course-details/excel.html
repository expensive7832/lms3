MICROSOFT EXCEL

Learn how to harness the power of Excel for data analysis, reporting, and problem-solving. This syllabus will take you from Excel beginners to advanced users, equipping you with the tools needed for real-world data challenges covering practical skills that help to manipulate data, perform calculations, create visualizations, and automate tasks with advanced tools.
